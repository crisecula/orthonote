OrthoNote

Gestión del conocimiento aplicado a análisis de datos clínicos en Traumatología

Este trabajo tiene como objetivo proporcionar una herramienta capaz de ayudar a la toma de decisiones por parte de cirujanos ortopédicos y traumatológicos.

Se proporcionará una aplicación en la nube con una interfaz preparada para introdcción de datos por parte de los médicos cumpliendo los estándares de las escalas usadas en Traumatología. Estos datos formarán una base de datos aplicada que será consultada por el sistema para obtener informes.

Las consultas a la base de datos se realizará mediante querys correspondientes a un banco de preguntas frecuentes que estarán predefinidas y también se proporcionará una interfaz para relizar querys sencillas por parte de los médicos.
